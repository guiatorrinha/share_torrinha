# Template da Funcionalidade Compartilhar

## Requisitos
- Nome da cidade franqueada
- Link AppLink
- Link Apple Store
- Link Google Play Store
- Link do repositório no GitHub

# Como usar

## Clonar repositório 
[Siga o passo a passo](/src/guides/createRepository.md)

## Criar arquivo
````sh
    ./meExecuteCriar.sh
````

## Fazer upload no GitHub
````sh
    ./meExecuteBuildar.sh
````

## Deploy
````sh
    yarn predeploy
````
````sh
    yarn deploy
````
